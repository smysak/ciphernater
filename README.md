# ciphernater

ciphernater.rb:
This program will receive a string of text, and 'enciphernate' it using a right-shifted caesar cipher.
Giving the program a negative integer will produce a left-shifted caesar cipher.

ciphernater_file.rb:
This variant will receive a multi-line input of text, and export it 'enciphernated', as "enciphernated_message.txt"
